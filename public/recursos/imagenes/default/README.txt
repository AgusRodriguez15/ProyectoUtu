Nota: Las vistas HTML ya usan un SVG inline como imagen por defecto,
por lo que no es estrictamente necesario tener un archivo físico.

Si deseas usar un PNG personalizado:
- Crea un archivo "user-default.png" (recomendado: 150x150px o 200x200px)
- Actualiza las referencias en los archivos HTML si es necesario
